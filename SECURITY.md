# Security Policy

## Supported Versions

Security fixes are provided for the latest published release.

## Reporting A Vulnerability

Please report suspected vulnerabilities privately to
[support@tinyopsstudio.com](mailto:support@tinyopsstudio.com). Include the
affected version, reproduction steps, and expected impact. Do not include
credentials or other sensitive data in a public issue.

TinyOps Studio will acknowledge a report within five business days and will
coordinate disclosure after a fix is available.
